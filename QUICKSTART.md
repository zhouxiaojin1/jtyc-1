# 快速启动指南

## 1. 启动应用

### Windows用户

双击运行 `run_app.bat`

或在命令行中执行：
```bash
conda activate ap
streamlit run app.py
```

### Linux/Mac用户

```bash
conda activate ap
streamlit run app.py
```

## 2. 访问应用

浏览器会自动打开，或手动访问：
```
http://localhost:8501
```

## 3. 使用流程

### 步骤1：数据预处理
1. 点击侧边栏的"📊 数据预处理"
2. 在"数据加载"选项卡中选择数据集（如 milano_traffic_nid.csv）
3. 点击"加载数据"按钮
4. （可选）进行缺失值处理和异常检测
5. 使用数据探索功能查看数据特征

### 步骤2：模型训练
1. 点击侧边栏的"🤖 模型训练"
2. 选择一个模型（建议新手先用LightGBM）
3. 调整参数：
   - 预测步数：1008（7天）
   - 历史窗口：2016（14天）
4. 点击"开始训练"按钮
5. 等待训练完成（LightGBM约2-5分钟）

### 步骤3：查看结果
1. 点击侧边栏的"📈 结果分析"
2. 在"预测结果"选项卡查看预测数据和指标
3. 选择区域进行可视化
4. 查看评估指标（MAE、RMSE、MAPE）

### 步骤4：对比模型（可选）
1. 重复步骤2，训练其他模型
2. 在"模型对比"选项卡对比不同模型的性能
3. 选择最佳模型

## 4. 常用操作

### 查看系统信息
- 点击"⚙️ 系统设置"
- 在"系统信息"选项卡查看环境配置

### 清理文件
- 点击"⚙️ 系统设置" > "文件管理"
- 使用清理按钮删除旧的输出文件

### 导出结果
- 在"结果分析"页面使用下载按钮
- 或直接在 `output/` 目录查找文件

## 5. 故障排除

### 应用无法启动
```bash
# 确认conda环境
conda activate ap

# 重新安装streamlit
pip install --upgrade streamlit
```

### 数据加载失败
- 确认数据文件在 `dataset/` 目录下
- 检查文件格式是否为CSV

### 训练失败
- 查看训练日志中的错误信息
- 在系统设置中运行健康检查
- 确认所有依赖包已安装

## 6. 获取帮助

- 查看完整文档：README_UI.md
- 运行系统健康检查：系统设置 > 关于 > 运行健康检查
- 查看训练日志：模型训练时展开"训练日志"

3. 查看预测结果
